# Automated-mail-generation-vba-uipath

Overview
This project automates email report generation and distribution by extracting data from SharePoint, processing it with Excel VBA, and sending formatted reports via Outlook. The solution is integrated with UiPath to enable hourly execution with minimal user interaction.

Business Value
•	Eliminates manual report preparation
•	Ensures timely and accurate email delivery
•	Reduces human error in repetitive reporting tasks
•	Improves operational efficiency

Workflow
1.	Data extracted automatically from SharePoint
2.	Excel VBA refreshes, processes, and formats data
3.	Two Excel reports generated dynamically
4.	Reports sent via Outlook to defined recipients
5.	UiPath triggers the process hourly or on demand

Tools & Technologies
•	Excel VBA
•	UiPath
•	SharePoint
•	Microsoft Outlook

Key Features
•	One-click and scheduled execution
•	Automated report formatting
•	Multi-file email attachments
•	Scalable for additional reports
